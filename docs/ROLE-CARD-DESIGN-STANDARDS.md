# 角色卡設計規範與品質標準

## ⚠️ 核心原則

**540 張角色卡每一張都必須獨立精心設計，嚴禁批次生成或套用模板。**

## 🎨 設計要求

### 每張角色卡必須具備的獨特性

1. **場景設定 (Scene)**
   - 不同的場景位置組合
   - 獨特的道具選擇和排列
   - 不同的時間設定（清晨/午後/黃昏/夜晚/深夜）
   - 天氣變化（晴天/陰天/雨後/霧氣/雪天）
   - 季節感（春櫻/夏荷/秋楓/冬雪）

2. **服裝細節 (Costume)**
   - 獨特的色彩搭配（主色+輔色+點綴色）
   - 不同的材質組合（絲綢/紗/絨/綢緞/薄紗）
   - 獨特的配飾選擇（頭飾/項鍊/手飾/腰飾）
   - Layer 1-10 各層都要有獨特設計
   - 符合分類視覺 DNA 但有創意變化

3. **動作姿態 (Action)**
   - 不同的姿勢（站立/坐姿/半躺/回眸/舞動/靜坐）
   - 獨特的手勢（持物/托腮/撫髮/執扇/捧花）
   - 不同的眼神（對視/遠望/低垂/側望/凝視）
   - 獨特的表情（淺笑/神秘/憂鬱/冷豔/溫柔/威嚴）

4. **光線氛圍 (Lighting)**
   - 不同的主光源位置和強度
   - 獨特的輔助光組合
   - 不同的環境光色溫
   - 特殊光效（月光/燭光/霞光/星光/靈火）
   - 氛圍營造（朦朧/清晰/戲劇性/柔和/強烈）

5. **個性特色 (Character)**
   - 每張卡都要有故事感
   - 標題要有詩意和畫面感
   - 整體氛圍要有獨特性格
   - 符合分類但有個人風格

## 📋 設計流程

### 單張角色卡開發時間：15-20 分鐘

1. **構思階段 (3-5 分鐘)**
   - 檢視分類 visualDNA
   - 想像獨特的場景畫面
   - 構思角色性格與故事

2. **設計階段 (8-10 分鐘)**
   - 編寫詳細的場景描述
   - 設計 10 層服裝系統
   - 設定動作與表情
   - 配置光線與氛圍

3. **檢查階段 (2-3 分鐘)**
   - 確認符合分類 visualDNA
   - 檢查與已有角色卡的差異性
   - 驗證 5 個獨特性要素
   - 確認標題有詩意

4. **記錄階段 (2 分鐘)**
   - 記錄已使用的場景/服裝/動作組合
   - 避免後續重複

## ✅ 品質檢查清單

### 每張角色卡完成後檢查：

- [ ] 場景設定與前面的卡不重複
- [ ] 服裝顏色搭配是新的組合
- [ ] 動作姿態有獨特性
- [ ] 光線氛圍有變化
- [ ] 標題有詩意和畫面感
- [ ] 符合分類的視覺 DNA
- [ ] 有獨特的故事感和性格
- [ ] 10 層服裝系統完整
- [ ] 禁止元素已避開

### 每 20 張角色卡完成後檢查：

- [ ] 20 張卡之間有明顯多樣性
- [ ] 場景類型分佈均勻
- [ ] 色彩搭配有豐富變化
- [ ] 動作姿態覆蓋多種類型
- [ ] 光線氛圍有冷暖變化
- [ ] 沒有重複的組合

## 🚫 禁止事項

❌ **嚴禁以下行為：**
- 批次生成 JSON
- 使用固定模板套用
- 複製貼上後只改標題
- 場景/服裝/動作大量重複
- 偷懶使用通用描述
- 忽略獨特性要求

## 📊 多樣性指標

### 單一分類內 20 張角色卡應達到：

- **場景位置**：至少 15 個不同位置
- **主色調**：至少 8 種不同色系
- **姿態類型**：至少 12 種不同姿勢
- **時間設定**：涵蓋晨/午/黃昏/夜/深夜
- **光線類型**：至少 6 種不同光源組合
