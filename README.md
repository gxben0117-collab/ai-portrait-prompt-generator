## 🎯 核心特性

### 五層模組化引擎
1. **Layer 1: Face Lock** (最高權重) - 保留原始臉部特徵
2. **Layer 2: Scene** (可變) - 分類專屬場景構建
3. **Layer 3: Costume** (可變) - 10層真實服裝物理
4. **Layer 4: Photography** (固定/半固定) - 專業攝影參數
5. **Layer 5: Visual Style** (可變) - 三選一風格系統

### 視覺 DNA 系統
每個分類都有獨特的「視覺基因」：
- 核心氛圍
- 色彩基因
- 標誌性道具
- 服裝風格
- 身材要求
- 場景類型
- 光線偏好
- 禁止元素

### 角色卡設計原則 ⚠️ 重要
**540 張角色卡每一張都必須獨立精心設計，嚴禁批次生成或套用模板。**

每張角色卡都要有獨特的：
- ✨ **場景設定** - 不同的場景位置、道具組合、時間天氣
- 👗 **服裝細節** - 獨特的色彩搭配、材質、配飾組合
- 💃 **動作姿態** - 不同的姿勢、手勢、眼神、表情
- 💡 **光線氛圍** - 獨特的光線組合、色溫、氛圍營造
- 🎭 **個性特色** - 每張卡都要有自己的故事感和性格

**品質標準**：
- 每張角色卡開發時間：15-20 分鐘
- 參考視覺基因，但創造獨特變化
- 避免重複的場景/服裝/動作組合
- 每 20 張卡互相檢查，確保多樣性
