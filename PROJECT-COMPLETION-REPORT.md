## 🎯 技術亮點

### 五層模組化引擎
```
Layer 1: Face Lock (1.5x) - 最高優先級保留臉部
Layer 2: Scene (1.2x) - 分類專屬場景
Layer 3: Costume (1.3x) - 10層服裝物理
Layer 4: Photography (1.1x) - 攝影參數
Layer 5: Style (1.0x) - 視覺風格
```

### 視覺 DNA 系統
每個分類包含：
- 核心氛圍 (atmosphere)
- 色彩基因 (colors)
- 標誌性道具 (iconicProps)
- 服裝風格 (costumeStyle)
- 身材要求 (bodyRequirements)
- 場景類型 (sceneTypes)
- 光線偏好 (lighting)
- 禁止元素 (prohibitions)

### Prompt 優化
- Token 數量優化（目標 120-150）
- 去重與衝突檢測
- 品質驗證報告
- 分層權重控制
